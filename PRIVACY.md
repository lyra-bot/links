# Lyra Bot — Privacy Policy

*Last updated: 28/02/2026*

## 1. Introduction
This Privacy Policy describes how Lyra ("the Bot") handles data. 
We are committed to protecting your privacy.

## 2. Data We Collect

### Stored Data
The Bot stores the following per-server configuration in a local file:
- **Server (Guild) ID** — to identify which server settings belong to
- **Language preference** — chosen display language (English/Russian)
- **DJ Role ID** — the role designated to control the DJ panel
- **Auto-disconnect timeout** — the configured timeout duration

### Temporary Data (not stored)
- **Voice channel membership** — checked in real-time to determine 
  when to auto-disconnect. Never logged or stored.
- **Music prompts** — sent to Google Lyria API for music generation. 
  Not stored after the session ends.
- **Command interactions** — processed in memory and not logged.

## 3. Data We Do NOT Collect
- Message content
- User messages or DMs
- User audio or voice data
- User IDs, usernames, or avatars
- Server member lists
- Any personally identifiable information (PII)

## 4. Third-Party Services
The Bot uses the **Google Lyria API** to generate music. Music prompts 
you provide are sent to Google's servers for processing. Please refer 
to [Google's Privacy Policy](https://policies.google.com/privacy) for 
their data handling practices.

## 5. Data Storage
Server settings are stored in a local JSON file on the bot's host 
server. No data is transmitted to external databases or analytics services.

## 6. Data Deletion
- Server settings are automatically removed when the bot is removed 
  from a server.
- You can reset settings at any time using the `/settings` command.
- To request complete data deletion, contact us at https://discord.gg/xsWbkpWF9g.

## 7. Children's Privacy
The Bot does not knowingly collect data from children under 13. 
The Bot is intended for use in compliance with Discord's own 
Terms of Service, which require users to be at least 13 years old.

## 8. Changes
We may update this policy at any time. Changes will be reflected 
in the "Last updated" date above.

## 9. Contact
For privacy-related questions: https://discord.gg/xsWbkpWF9g
